Learning the Unix Way

I have recently learned some basic unix commands. So sharing a bit of it through my first technical blog.Just giving you a brief idea here about how vi command works.

Vi Command

vi command launches a editor. It is basically used to edit file. Using vi you can edit the existing file or if file doesn’t exist it will create new file.If you use vi without file name it will open new file where you can enter the text and edit and while coming out you can enter a valid name and save the text. It has two main modes of operation: command mode, and insert mode.
i)Command Mode : After a file is opened it is default opened in command
mode , and you will not see the words you are typing on the screen.Input from the keyboard will be consider it as vi commands
ii)Insert Mode : To enter the text you have to put vi in insert mode by pressing ‘i’ or ‘a’  after  which you can add the text and whatever is being type will be seen on the  screen. . To switch between these mode Esc key is used.

Examples :vi myfile.txt
It will edit the file myfile.txt.
:wq Typing the above command would save the changes you have made and exit the vi editor
We can have different option to exit vi :
1) Quit without saving : If you don’t  want to save the work :q  will exit without saving  your editing in vi.
2) Write & quit : . Simple :w saves the current file but don’t exit. For save and quit  :wq is  used in vi.
3) Forced Quite : Exclamation  sign at the end of  exit commands ( :q! , :wq! )  causes a forced  quit from vi  after ignoring editing (for :q!)  or writing (for :wq!)  all the change.
